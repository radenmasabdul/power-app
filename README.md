## ✨ About the Project

Power App is a landing page project created to practice UI slicing from design into a modern, responsive website.

This project focuses on:
- Clean and structured layout (hero, about, services, products)
- Responsive design for mobile and desktop devices
- UI components such as navbar, cards, and subscription form
- Simple animations and basic interactions

---

## 🚀 Features

- Responsive navbar with mobile menu
- Hero section with split layout
- About section with company logos
- Testimonial / services section
- Product / services grid layout
- Email subscription input UI
- Clean and modern design system

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla / optional jQuery)
- Google Fonts (Poppins & Inter)

---

## 📋 Prerequisites

Before running Power App locally, make sure you have installed:

- **A modern browser** (Chrome, Edge, Firefox, etc.)
- **Git** (to clone the repository)
- No build tools required (this is a static project)

---

## ⚡ Quick Start

### 1. Clone the Repository
```
git clone https://github.com/radenmasabdul/mampu-app.git
cd mampu-app
```

### 2. Open the Project
```
Simply open index.html in your browser.

OR use Live Server (recommended):

Install Live Server extension in VS Code
Right click index.html
Click "Open with Live Server"
```

## 📱 Responsive Design
- Desktop-first layout approach
- Mobile-friendly adjustments for all sections
- Grid layouts collapse into single column on small screens
- Hero and content sections adapt to vertical stacking

## 📂 Project Structure

